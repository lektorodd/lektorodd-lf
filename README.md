# Lektorodd-lf Format

Lite custom-format for å eksportera løysingsforlaga på bloggen til ok. pdf-ar...

> [!NOTE]
> Overskriftnivå:
> `#` for "Del 1" og "Del 2"
> `##` for "Oppgåve 1" osb
> `###` for "a) løyse likning" osb  

**OBS** Inkluder 

```md
::: {.content-hidden unless-format="html"}
dette gjerdet
:::
```

rundt ting som berre skal vera med på nettsida.
